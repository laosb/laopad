#  Laopad

A simple application for you to use Trackpad as Num Pad.

## Usage

1. Open this util.
2. Place your cursor in the window.
3. Type on your Trackpad.

```
7 8 9 Del
4 5 6
1 2 3 Clr
, 0 . Enter
```

(Just imagine your trackpad divided into a 4x4 grid above and type ahead.)
